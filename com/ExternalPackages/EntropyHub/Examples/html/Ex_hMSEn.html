Syntax:

[MSx, Sn, Ci] = hMSEn(Sig, 'Scales', 3, 'RadNew', 0, 'Plotx', false)
  • Sig Time series signals, a vector of length > 10.
  • Scales Number of grained time scales, a positive integer.
  • RadNew Radius rescaling method, an integer in the range [0 4]. When the Base entropy method speci ed by Mobj is SampEn or ApEn, RadNew allows the radius threshold to be updated based on the grained signal at each time scale (). If a radius threshold value (r) is speci ed in Mobj, this becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew speci es one of the following methods: 0 - no rescaling / 1 - Standard Deviation / 2 - Variance / 3 - Mean Absolute Deviation / 4 - Median Absolute Deviation
  • Plotx A plot of the multiscale entropy curve: true - Plots time scale vs entropy value / false - No plot.
  • MSx Multiscale entropy estimate at each time scale, a vector of length Scales.
  • Sn Average entropy value across each scale of hierarchical tree, a vector of length Scales.
  • Ci Complexity index (area under the multiscale entropy curve), a scalar.

References:

  1. Ying Jiang, C-K. Peng and Yuesheng Xu, Hierarchical entropy analysis for biological signals, Journal of Computational and Applied Mathematics 236.5 (2011): 728-742.

About hMSEn():

To use the hierarchical multiscale entropy function (hMSEn), one must first create a multiscale entropy object using MSobject() to specify the type of entropy and its parameters used in the multiscale entropy estimation.
hMSEn() returns three results:
  1. MSx the entropy estimate at each time scale, a vector of length
  2. Sn the average entropy value across all nodes at each temporal scale, a vector of length Scales
  3. Ci the complexity index (area under the multiscale entropy curve), a scalar value

The 'RadNew' Argument:

Each multiscale entropy function accepts a 'RadNew' name/value pair argument to perform radius distance threshold rescaling at each grained time-scale if the Base entropy method specified by Mobj is SampEn or ApEn.
If a radius threshold value (r) is specified in Mobj, RadNew becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods:
0 No rescaling
1 Standard deviation -
2 Variance -
3 Mean Abs Deviation -
4 Median Abs Deviation -

Note:

In the following examples, example signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform' uniformly distributed random numbers in range [0 1]
  • 'gaussian' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers' uniformly distributed pseudorandom integers in range [1 8]
  • 'chirp' chirp signal with the following parameters: f0 = .01, t1 = 4000, f1 = .025
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

Example 1:

Import a signal of uniformly distributed pseudorandom integers in the range [1,8] and create a multiscale entropy object with the following parameters:
EnType = K2En(), embedding dimension = 3, radius threshold = 3
X = ExampleData('randintegers');
Mobj = MSobject('K2En', 'm', 3, 'r', 3)
Mobj = struct with fields:
Func: @K2En m: 3 r: 3
Calculate the hierarchical multiscale Kolmogorov entropy over 3 temporal scales.
MSx = hMSEn(X, Mobj)

Example 3:

Import a chirp signal and create a multiscale entropy object with the following parameters:
EnType = ApEn(), embedding dimension = 4, radius threshold = 1.25
X = ExampleData('chirp');
Mobj = MSobject('SampEn', 'm', 4, 'r', 1.25)
Mobj = struct with fields:
Func: @SampEn m: 4 r: 1.2500
Calculate the hierarchical multiscale sample entropy over 4 temporal scales where the radius distance threshold value (r) specified by Mobj becomes scaled by the mean absolute deviation of the signal at each node in the hierarchical tree. In addition to the average entropy at each scale (Sn) and the complexity index (Ci), return a plot of the multiscale entropy curve and the hierarchical tree with the entropy value at each node.

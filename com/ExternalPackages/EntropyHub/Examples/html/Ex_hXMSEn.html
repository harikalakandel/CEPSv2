Syntax:

[MSx, Sn, Ci] = hXMSEn(Sig, 'Scales', 3, 'RadNew', 0, 'Plotx', false)
  • Sig Time series signals, a N x 2 matrix where N > 10
  • Scales Number of grained time scales, a positive integer.
  • RadNew Radius rescaling method, an integer in the range [0 4]. When the Base entropy method speci ed by Mobj is XSampEn or XApEn, RadNew allows the radius threshold to be updated based on the grained signal at each time scale (). If a radius threshold value (r) is speci ed in Mobj, this becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew speci es one of the following methods: 0 - no rescaling / 1 - Standard Deviation / 2 - Variance / 3 - Mean Absolute Deviation / 4 - Median Absolute Deviation
  • Plotx A plot of the multiscale entropy curve: true - Plots time scale vs entropy value / false - No plot.
  • MSx Multiscale cross-entropy estimate at each time scale, a vector of length Scales.
  • Sn Average cross-entropy value across each scale of hierarchical tree, a vector of length Scales.
  • Ci Complexity index (area under the multiscale entropy curve), a scalar.

References:

  1. Ying Jiang, C-K. Peng and Yuesheng Xu, Hierarchical entropy analysis for biological signals, Journal of Computational and Applied Mathematics 236.5 (2011): 728-742.

About hXMSEn():

To use the hierarchical multiscale cross-entropy function (hXMSEn), one must first create a multiscale entropy object using MSobject() to specify the type of entropy and its parameters used in the multiscale entropy estimation.
hXMSEn() returns three results:
  1. MSx the cross-entropy estimate at each time scale, a vector of length
  2. Sn the average cross-entropy value across all nodes at each temporal scale, a vector of length Scales
  3. Ci the complexity index (area under the multiscale entropy curve), a scalar value

The 'RadNew' Argument:

Each multiscale cross-entropy function accepts a 'RadNew' name/value pair argument to perform radius threshold rescaling at each grained time-scale if the Cross entropy method specified by Mobj is XSampEn or XApEn.
If a radius distance threshold value (r) is specified in Mobj, RadNew becomes the rescaling coefficient, otherwise it is set to 0.2 (default). The value of RadNew specifies one of the following methods:
0 No rescaling
1 Standard deviation -
2 Variance -
3 Mean Abs Deviation -
4 Median Abs Deviation -

Note:

In the following examples, signals / data are imported into MatLab using the ExampleData function from EntropyHub.
To use this function as outlined in the examples below, an internet connection is required.
ExampleData accepts any of the following strings:
  • 'uniform2' uniformly distributed random numbers in range [0 1]
  • 'gaussian2' normally distributed random numbers with mean = 0, and standard deviation = 1
  • 'randintegers2' uniformly distributed pseudorandom integers in range [1 8]
  • 'lorenz' X, Y, Z components of the Lorenz system (sigma: 10, beta: 8/3, rho: 28) [Xo = 10, Yo = 20, Zo = 10]
  • 'henon' X, Y components of the Henon attractor (alpha: 1.4, beta: .3) [Xo = 0, Yo = 0]

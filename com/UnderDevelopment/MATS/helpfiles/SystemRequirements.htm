System Requirements

 

To use the complete functionality of MATS the user need to have matlab version 6.5 or later and the toolboxes of Signal Processing, System Identification and Statistics. Otherwise some of the functions of MATS might not run.

To run MATS, the user has either to move to the directory where MATS and all the relevant files are or alternatively to add this directory to the matlab paths (use 'addpath' on the command line, or from the menu in newer versions File -> Set path ... -> Add folder with subfolders ...).

MATS makes limited use of external files (k-d-tree for the search of neighboring points). External files (.dll for windows and .mexglx for Linux) are supported by matlab versions 6.5 or later.

Some operations in MATS may be slow depending on the system memory. Especially, the computation of correlation dimension may give stop signaling "out-of-memory" if the time series is very large (we experienced this problem in PC Core2Duo 3.17GHz and 2GB RAM).
